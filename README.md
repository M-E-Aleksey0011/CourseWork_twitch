Добро пожаловать на страницу моего проекта!

Это страница моей кроссовой работы — разработка Twitch-бота с управлением через Telegram.

В основе проекта лежит Twitch-бот, использующий нейросеть DeepSeek через API DeepSeek, а управление ботом осуществляется с помощью Telegram-бота.

Ключевые особенности проекта:

Возможность добавления ключей API DeepSeek прямо через Telegram, что значительно упрощает работу.

Несколько дополнительных удобных функций, которые вы сможете обнаружить в Telegram-боте.

Простой и понятный доступ к базе данных: для полноценной работы бота достаточно правильно настроить базу данных, которая уже подготовлена, просто вставив необходимые данные.

В общем, по сути это Twitch-бот с управлением через Telegram, и для его использования всё, что вам нужно — это корректно настроить базу данных.
